# VisitHosp

>  Transformando a incerteza em cuidado e previsibilidade.

---

## 📝 Descrição do Projeto

O Visit Hosp é um sistema inteligente de gestão de fluxos para hospitais e seus visitantes

Objetivo: Sincronizar, em tempo real, a agenda de visitas com o status clínico do paciente e a lotação real da ala

Problema: Resolve a "viagem perdida" (visitas em horários de exames/banho), riscos de infecção (via triagem digital), falhas de segurança e exposição de dados sensíveis (LGPD)

Público-alvo: Hospitais de média e alta complexidade (B2B) e familiares que buscam uma jornada de visitação segura e previsíve]

---

## 🛠️ Tecnologias e Stacks Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **Frontend:** HTML, CSS, JS
* **Backend:** Flask (Python)  
* **Banco de Dados:** MySQL Workbench
* **Ferramentas & Outros:** Docker, Figma, N8N, Google Maps API, Bibliotecas de Criptografia, Draw.io
